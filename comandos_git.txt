git init . -> Cria um repositorio git na pasta atual
git status -> Mostra o status do trabalho de cada arquivo.
git add -> Adiciona aquivo para ser comitado.
git commit -m "msg" -> Faz commit do ue estava para ser comitado.
git log -> Mostra todo o histórico de commit.

git config - -global user.name “Seu nome”
git config - -global user.email “seu@email”
